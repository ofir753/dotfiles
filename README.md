# Ofir's DotFiles

## Install
```bash
sudo echo a && git clone https://github.com/ofir753/dotfiles.git && cd dotfiles && sudo echo a && ./install
```

Incase you are using windows terminal add `windows_terminal_binds.json` to your windows terminal
